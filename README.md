# archive-indexportal-hth

## 项目简介

`archive-indexportal-hth` 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定网站、域名或服务，仅作为 HTML 页面内容的结构化存档与索引展示用途。

所有页面均为静态 HTML，可直接通过 GitHub Pages 或其他静态托管服务访问。

---

## 目录结构说明

```
archive-indexportal-hth/
├── index.html          # 入口页面，提供归档索引
├── pages/              # 存放归档的 HTML 页面文件
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 页面引用的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，通常包含归档列表或导航链接。
- `pages/`：按归档顺序存放独立的 HTML 页面文件。
- `assets/`：页面依赖的样式、脚本或图片等资源。

---

## 页面归档说明

- 页面均为独立完整的 HTML 文件，可直接在浏览器中打开。
- 页面内容可能包含文字、表格、链接、图片等常见网页元素。
- 归档页面不涉及任何形式的用户数据收集、登录或交互功能。
- 页面命名规则：`page-{序号}.html`，序号从 001 开始递增。

---

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/archive-indexportal-hth.git
   ```
2. 在浏览器中打开 `index.html` 即可浏览归档内容。
3. 也可通过 GitHub Pages 发布后在线访问。
